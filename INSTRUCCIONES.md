# 📱 Cómo Compartir la Invitación por WhatsApp

## Opción 1: Usar un Servidor Web Gratuito (Recomendado)

Para que la invitación se vea perfecta cuando la compartas por WhatsApp, necesitas subirla a un servidor web. Aquí tienes opciones gratuitas:

### 1. **Netlify** (Más fácil)
1. Ve a [netlify.com](https://www.netlify.com)
2. Arrastra la carpeta `JoaquiCumple` a la página
3. Obtendrás una URL como: `https://tu-invitacion.netlify.app`
4. ¡Copia esa URL y compártela por WhatsApp!

### 2. **Vercel**
1. Ve a [vercel.com](https://vercel.com)
2. Sube la carpeta del proyecto
3. Obtendrás una URL automáticamente

### 3. **GitHub Pages**
1. Crea una cuenta en GitHub
2. Crea un repositorio nuevo
3. Sube el archivo `index.html`
4. Activa GitHub Pages en la configuración
5. Tu URL será: `https://tu-usuario.github.io/tu-repositorio`

## Opción 2: Usar un Servicio de Alojamiento Temporal

- **Surge.sh**: Gratis y muy fácil de usar
- **Firebase Hosting**: Gratis con buena velocidad
- **000webhost**: Hosting gratuito simple

## ⚠️ Importante

**NO puedes compartir archivos locales** (file://) por WhatsApp y que se vean bien. Necesitas una URL web (https://).

## 📝 Pasos Rápidos con Netlify:

1. Ve a: https://app.netlify.com/drop
2. Arrastra toda la carpeta `JoaquiCumple`
3. Espera unos segundos
4. Copia la URL que te dan
5. ¡Comparte esa URL por WhatsApp!

## ✨ Características Agregadas para WhatsApp:

- ✅ Meta tags optimizados para WhatsApp
- ✅ Vista previa mejorada cuando compartes el enlace
- ✅ Optimizado para móviles
- ✅ Se ve igual en todos los dispositivos
- ✅ Animaciones suaves y efectos visuales

## 🎯 Una vez que tengas la URL:

Simplemente copia y pega la URL en WhatsApp. Los contactos verán:
- Una vista previa bonita con el título
- La descripción de la invitación
- Al abrir, verán la invitación completa con todos los efectos

¡Listo! 🎉
